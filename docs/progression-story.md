# Epic → Epic 3.0 → Anthem Training Progression

> Strategic Dimension 3: Progressive training pathway that respects the certification timeline.

## The Challenge

Anthem represents Honeywell's first clean-sheet avionics architecture in over two decades. The EPIC-to-Anthem transition is not a software update — it's an architectural generation change affecting every aspect of pilot interaction:

- **Touch-first interface** replacing physical knobs and buttons
- **Customizable display layouts** replacing fixed OEM arrangements
- **AI-driven PilotPredict** introducing predictive automation
- **Cloud connectivity** enabling real-time data sharing

73% of pilots have inadvertently selected wrong automation modes, and insufficient crew knowledge of automated systems contributed to over one-third of accidents and serious incidents.

## Progressive Training Strategy

### Phase 1: Epic Proficiency (Current Fleet)

**Target**: Pilots operating Primus Epic today
**Focus**: Establish baseline competency metrics using existing workflows

- Standard CBTA assessment on current Epic procedures
- Voice-based cognitive load baseline establishment
- Readback accuracy benchmarking with current frequencies/procedures
- No new interface concepts — purely assessment

### Phase 2: Epic 3.0 Bridge (2026 Launch)

**Target**: Pilots transitioning to Epic 3.0
**Focus**: Introduce Anthem-like features within familiar Epic architecture

- Touch interaction training alongside traditional controls
- PilotPredict introduction with trap detection drills
- Display customization familiarization
- Mixed-mode operation (touch + knobs)

### Phase 3: Anthem Transition

**Target**: Pilots moving to full Anthem flight deck
**Focus**: Complete touch-first operation with AI-assisted workflows

- Full touch-only cockpit drills
- Advanced PilotPredict integration drills
- Connected Mission Manager workflows
- Real-time data sharing and collaborative scenarios

## Competency Continuity

CBTA competency scores carry across all three phases:

| Competency | Epic | Epic 3.0 | Anthem |
|-----------|------|----------|--------|
| COM | Standard phraseology | Same + touch acknowledgment | Same + AI-assisted responses |
| WLM | Manual task management | Touch + knob hybrid | Touch-only + PilotPredict |
| SAW | Traditional scan patterns | Customizable display awareness | AI-enhanced situational display |
| KNO | Epic procedures | Epic 3.0 new features | Anthem architecture knowledge |
| PSD | Standard decision making | PilotPredict awareness | PilotPredict integration |
| FPM | FMS + autopilot | Touch FMS interaction | Connected Mission Manager |

## Certification Considerations

- Anthem is **not yet FAA-certified** as of March 2026
- No FSB activity, qualified FTD, or formal transition syllabus exists
- Training approach must be usable under Part 91/135 without AQP
- This prototype demonstrates the concept; regulatory pathway requires separate effort
